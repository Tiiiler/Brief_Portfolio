Ardriuno Uno Song Recorder (Sophmore Year)

This assignment's purpose was to create a program on a Ardriuno Uno board that would allow the user to record and store a short song segment. This program would allow the user to enter in keys on 4x4 keypad and the Uno board would take this input and convert each key to a different tone. You record a certain number of notes and then choose to stop recording and play the song out of a speaker also hooked to the board. You could save up to two songs at a time and switch back and forth to add notes to the end if you wanted to.

Skills Used:

  • C/C++ Language
  • Microprocessor components
  • Arduino Software

Producer-Consumer Problem (Junior Year)

This assignment's purpose was to solve the producer-consumer problem using basic operating system calls and functions. It obtained input from the user on how many messages, consumers and producers there were so that each the producer and consumer could read and write to eachother in a shared memory buffer. Each producer sending information or consumer recieving it creates a thread to execute its task. In addition to this, a mutex and semaphores are created to achieve synchronization between the two programs. In other words, each program knew when its turn to execute was. This project really helped show me how an operating system decides how tasks and threads get executed at a low level.

Skills Used:

  • C Language
  • Linux OS
  • Operating System Components

Smarter Slow Cooker (Senior Year)

For my senior project, we are to design and modify an off the shelf appliance to incoorporate the idea of the Internet of Things. My team and I are still in the process of completing this project, but we have our design completed. We are buying a standard low/medium/high slow cooker that we will add the following functionalities:

  • Variable temperature to the nearest 5 degrees fahrenheit
  • Temperature monitoring of the bowl
  • The addition of and temperature monitoring of a food probe
  • Wi-Fi connectivity
  • Mixing up the contents of the bowl
  • Setting/controlling a timer

All of these additions will be controllable from a Android application connected to Wi-Fi (remotely) and/or from the cooker itself (locally).
